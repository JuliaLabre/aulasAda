PASSO A PASSO PARA O FUNCIONAMENTO DO EXPRESS
- Criar a pasta ou repositório
	- express nome-do-projeto
		-cd nome-do-projeto
			- npm install
			- npm install express ejs (se quiser terocar o Jade pelo ejs)
			- npm install dotenv
			- npm i express-session cookie-parser
 			- npm i sequelize (gerenciador de banco de dados)
			- npm run start

Agora é configurar de acordo com a preferencia.
